# Software Architecture

Este repositório reúne estudos, conceitos e exemplos relacionados a **Software Architecture**, com foco em como estruturar aplicações de forma organizada, escalável e sustentável ao longo do tempo.

A arquitetura de software define **a estrutura de alto nível de uma aplicação**, incluindo camadas, componentes, responsabilidades e a forma como eles se comunicam.

## Tópicos abordados

- Princípios de arquitetura
- Arquitetura em camadas (Layered Architecture)
- Clean Architecture
- Arquitetura Hexagonal
- Arquitetura de Microsserviços
- Domain Driven Design (DDD)
- Modularização
- Gerenciamento de dependências

## Conceitos importantes

- Separação de responsabilidades
- Baixo acoplamento
- Alta coesão
- Inversão de dependência
- Definição de limites arquiteturais
- Evolução e manutenção de sistemas

## Tecnologias e ferramentas

Os exemplos podem incluir:

- Java
- Spring Boot
- APIs REST
- Docker
- Sistemas de mensageria

## Propósito

Este repositório tem como objetivo servir como **material de estudo e referência** para o entendimento e aplicação de arquiteturas modernas no desenvolvimento de sistemas.
